# Dino Encyclopedia
Aplikasi web ensiklopedia dinosaurus yang interaktif dengan fitur pemetaan fosil global. Proyek ini dibuat untuk memenuhi Tugas Akhir Mata Kuliah Pemrograman Web.

# Fitur
* **Peta Interaktif:** Visualisasi lokasi penemuan fosil pada peta dunia.
* **Ensiklopedia:** Detail lengkap spesies (Diet, Periode, Famili, Deskripsi).
* **Dashboard Admin:** Mode Admin untuk manajemen data (CRUD).
* **Manajemen Tag:** Pengaturan kategori Famili, Periode, dan Formasi.

# Cara Instalasi (How to Run)
Ikuti langkah berikut untuk menjalankan (XAMPP):
1.  Clone repository ini atau download ZIP-nya.
2.  Nyalakan Apache dan MySQL pada XAMPP.
3.  Buka `localhost/phpmyadmin`, buat database baru bernama **`dino_db`**.
4.  Klik tab **Import**, lalu pilih file **`dino_db.sql`** yang ada pada proyek ini.
6.  Klik **Go** untuk mengimpor tabel dan data.
7.  Buka browser dan akses: `http://localhost/dino-app`.
